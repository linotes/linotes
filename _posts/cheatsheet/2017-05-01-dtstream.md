### 统计 apache 的 access.log 中访问量最多的前 5 个 ip 地址

`access.log` 通常格式如下：

```
127.0.0.1 - peter [9/Feb/2017:10:34:12 -0700] "GET /sample-image.png HTTP/2" 200 1479
```

因此，仅处理第 1 字段。

```bash
$ sudo cat /var/log/httpd/test-access.log \
 | awk '{print $1}' \
 | sort \
 | uniq -c \
 | sort -rn \
 | head -5
```









### 如何查看二进制文件 file 的内容

```bash
$ hexdump -C file
```

`-C` 以十六进制和 ASCII 码显示
