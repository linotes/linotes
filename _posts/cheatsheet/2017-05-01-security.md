### SELinux



#### 检查状态

```bash
$ sestatus
```



#### 切换为放行模式

```bash
$ sudo setenforce Permissive
```



#### 禁用 SELinux

```bash
$ sudo setenforce 0
```

该方法仅在当前禁用，重启后恢复。

永远禁用：

```bash
$ sudo vi /etc/selinux/config

SELINUX=disabled
```


#### 让 SELinux 为网页流量放行：

```bash
$ sudo semanage permissive -a httpd_t
```
