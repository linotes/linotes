## CentOS 配置 DNS 服务

使用 bind 创建 DNS 服务。


DNS Server:

* OS : CentOS 7 minimal server
* Hostname : dns.zion.matrix
* IP Address : 192.168.1.5/24


### 安装 bind

```bash
$ sudo yum install bind bind-utils -y
```
